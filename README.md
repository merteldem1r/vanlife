# VANLIFE React App
Vanlife is a react project which I made with Tailwind CSS and used react, react router features. I got figma design file from Bob Ziroll on scrimba platform.

https://merteldem1r-vanlife.netlify.app/

Features: 
* React Router
* Relative Paths
* Search Parameters
* Components & Pages
* API
* Hooks & Custom Hooks
* Error Handling
* Tailwind CSS
